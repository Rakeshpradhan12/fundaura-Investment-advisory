# Fundaura Site (React + TypeScript + Vite)

## Run locally
```bash
npm install
npm run dev
```

## Build & preview
```bash
npm run build
npm run preview
```

## Tests
```bash
npm run test
```

## Deploy
Upload the `dist/` folder to Netlify or Vercel. For Netlify Forms, the forms are already marked with `data-netlify="true"`.
